# LaTeX数学公式测试

## 基础运算和符号

行内公式：$a^2 + b^2 = c^2$

平方和：$x_1^2 + x_2^2 + \cdots + x_n^2$

加法：$a + b = c$

乘法：$a \times b = c$

不等于：$a \neq b$

小于等于：$a \leq b$

大于等于：$a \geq b$

## 分数和根号

简单分数：$\frac{1}{2}$

复杂分数：$\frac{a+b}{c-d}$

平方根：$\sqrt{x}$

立方根：$\sqrt[3]{x}$

## 基础函数

三角函数：$\sin(x) + \cos(x) = 1$

指数函数：$e^x = \sum_{n=0}^{\infty} \frac{x^n}{n!}$

对数函数：$\log_a(b) = c$

自然对数：$\ln(x)$

## 微积分符号

导数：$\frac{dy}{dx} = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}$

偏导数：$\frac{\partial f}{\partial x}$

积分：$\int_a^b f(x) dx$

定积分：$\int_0^{\infty} e^{-x^2} dx = \frac{\sqrt{\pi}}{2}$

多重积分：$\int_0^1 \int_0^1 f(x,y) dx dy$

极限：$\lim_{x \to \infty} \frac{1}{x} = 0$

极限（导数定义）：$\lim_{h \to 0} \frac{f(x+h) - f(x)}{h}$

## 统计学公式

平均值：$\bar{x} = \frac{1}{n}\sum_{i=1}^{n} x_i$

方差：$\sigma^2 = \frac{1}{n}\sum_{i=1}^{n} (x_i - \bar{x})^2$

标准差：$\sigma = \sqrt{\frac{1}{n}\sum_{i=1}^{n} (x_i - \bar{x})^2}$

协方差：$\text{Cov}(X,Y) = \frac{1}{n}\sum_{i=1}^{n} (x_i - \bar{x})(y_i - \bar{y})$

## 几何公式

勾股定理：$a^2 + b^2 = c^2$

圆的面积：$A = \pi r^2$

圆的周长：$C = 2\pi r$

球的体积：$V = \frac{4}{3}\pi r^3$

球的表面积：$S = 4\pi r^2$

椭圆面积：$A = \pi ab$

## 物理公式

速度：$v = \frac{s}{t}$

速度（微分形式）：$v = \frac{ds}{dt}$

加速度：$a = \frac{dv}{dt} = \frac{d^2s}{dt^2}$

动能：$E_k = \frac{1}{2}mv^2$

动量：$p = mv$

万有引力：$F = G\frac{m_1 m_2}{r^2}$

爱因斯坦质能方程：$E = mc^2$

## 化学方程式

水的分解：$2H_2O \rightarrow 2H_2 + O_2$

燃烧反应：$CH_4 + 2O_2 \rightarrow CO_2 + 2H_2O$

酸碱中和：$H^+ + OH^- \rightarrow H_2O$

## 向量和矩阵

二维向量：$\vec{v} = \begin{pmatrix} x \\ y \end{pmatrix}$

三维向量：$\vec{a} = \begin{pmatrix} a_1 \\ a_2 \\ a_3 \end{pmatrix}$

行列式：$\det(A) = |A|$

单位矩阵：$I = \begin{pmatrix} 1 & 0 \\ 0 & 1 \end{pmatrix}$

## 复杂数学表达式

欧拉公式：$e^{i\pi} + 1 = 0$

傅里叶变换：$\hat{f}(\xi) = \int_{-\infty}^{\infty} f(x)e^{-2\pi i x \xi} dx$

泰勒展开：$f(x) = \sum_{n=0}^{\infty} \frac{f^{(n)}(a)}{n!}(x-a)^n$

## 特殊函数

伽马函数：$\Gamma(z) = \int_0^{\infty} t^{z-1}e^{-t} dt$

贝塞尔函数：$J_n(x) = \frac{1}{\pi}\int_0^{\pi} \cos(n\tau - x\sin\tau) d\tau$

误差函数：$\text{erf}(x) = \frac{2}{\sqrt{\pi}}\int_0^x e^{-t^2} dt$

## 不等式和逻辑

柯西-施瓦茨不等式：$|\langle x,y \rangle| \leq \|x\|\|y\|$

三角不等式：$|a+b| \leq |a| + |b|$

德摩根定律：$\overline{A \cup B} = \overline{A} \cap \overline{B}$

## 希腊字母

小写希腊字母：$\alpha, \beta, \gamma, \delta, \epsilon, \zeta, \eta, \theta, \iota, \kappa, \lambda, \mu, \nu, \xi, \pi, \rho, \sigma, \tau, \upsilon, \phi, \chi, \psi, \omega$

大写希腊字母：$\Alpha, \Beta, \Gamma, \Delta, \Epsilon, \Zeta, \Eta, \Theta, \Iota, \Kappa, \Lambda, \Mu, \Nu, \Xi, \Pi, \Rho, \Sigma, \Tau, \Upsilon, \Phi, \Chi, \Psi, \Omega$

## 特殊符号和运算符

无穷大：$\infty$

正负号：$\pm$

点乘：$\cdot$

属于：$\in$

子集：$\subset$

并集：$\cup$

交集：$\cap$

箭头：$\rightarrow, \leftrightarrow, \Rightarrow, \Leftrightarrow$

## 上下标示例

单字符上下标：$x_1^2, e^x, H_2O, CH_4$

多字符上下标：$x_{min}^{max}, e^{i\pi}, \sum_{i=1}^{n}$

混合上下标：$a_{ij}^{kl}, x_{t_1}^{t_2}$
